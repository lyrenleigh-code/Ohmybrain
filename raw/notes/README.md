# notes/ — 对话记录、会议笔记与个人思考

## 存放内容
- Claude Code 对话中产生的重要记录
- 会议纪要
- 个人灵感、思考、复盘
- 系统设计文档和计划

## 来源
- 手动整理
- Claude Code 对话导出

## 命名规范
```
YYYY-MM-DD-主题简称.md
```

示例：
```
2026-04-12-my-brain-setup-plan.md
2026-04-12-toolchain.md
2026-04-12-weekly-review.md
```

## 文件结构建议
```markdown

- **类型**：对话记录 / 会议纪要 / 个人思考 / 设计文档
- **参与者**：（如适用）

```
