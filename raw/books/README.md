# books/ — 书籍笔记与章节摘要

## 存放内容
- 读书笔记
- 章节摘要
- Readwise 导出的书籍高亮

## 来源工具
- **Readwise**：电子书高亮导出
- 手动：阅读后整理

## 命名规范
```
YYYY-MM-DD-书名简称.md                    # 整本书笔记
YYYY-MM-DD-书名简称-chNN.md               # 按章节拆分
```

示例：
```
2026-04-12-thinking-fast-and-slow.md
2026-04-12-designing-data-intensive-apps-ch05.md
```
