# courses/ — 课程讲义与学习笔记

## 存放内容
- 在线课程笔记（Coursera、MIT OCW、YouTube 系列等）
- 讲义整理
- 作业和练习记录

## 来源工具
- 手动整理
- 视频转录（配合 Whisper / Firecrawl）

## 命名规范
```
课程名简称/                               # 每门课一个子目录
├── YYYY-MM-DD-lecture-NN-主题.md          # 单节课笔记
├── YYYY-MM-DD-summary.md                 # 课程总结
└── YYYY-MM-DD-homework-NN.md             # 作业记录
```

示例：
```
cs231n/
├── 2026-04-12-lecture-01-intro.md
├── 2026-04-15-lecture-02-image-classification.md
└── 2026-04-20-summary.md
```
