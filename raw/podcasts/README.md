# podcasts/ — 播客转录文本

## 存放内容
- 播客节目转录
- 音频访谈转录

> **只存转录后的 markdown/txt，不存音频原件。**

## 来源工具
- **Whisper**：下载音频后本地转录
- **VOMO AI**：在线上传处理

## 命名规范
```
YYYY-MM-DD-播客名-期数或主题.md
```

示例：
```
2026-04-12-lex-fridman-ep400-sam-altman.md
2026-04-12-hardcore-history-ep67.md
```

## 文件结构建议
```markdown
# 播客标题 — 期数

- **节目**：
- **嘉宾**：
- **日期**：
- **时长**：
- **来源**：URL
- **转录工具**：Whisper / VOMO AI

---

（转录正文）
```
