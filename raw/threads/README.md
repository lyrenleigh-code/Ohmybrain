# threads/ — 社交媒体长帖与推文串

## 存放内容
- X (Twitter) 长推文串
- LinkedIn 深度帖子
- Reddit 高质量讨论帖
- Hacker News 讨论

## 来源工具
- **Firecrawl**：抓取页面为 markdown
- 手动：复制整理

## 命名规范
```
YYYY-MM-DD-作者-主题简称.md
```

示例：
```
2026-04-12-karpathy-llm-training-tips.md
2026-04-12-simonw-prompt-engineering.md
```

## 文件结构建议
```markdown
# 帖子标题 / 主题

- **作者**：
- **平台**：X / LinkedIn / Reddit / HN
- **来源**：URL
- **日期**：

---

（帖子全文）
```
