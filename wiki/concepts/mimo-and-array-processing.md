# MIMO 与阵列处理

## 定义

MIMO（多输入多输出）与阵列处理方向研究利用多个发射/接收换能器（阵元）进行空间域信号处理的技术。在水声通信中，MIMO 技术可以通过空间复用提高传输速率，或通过空间分集提高通信可靠性。阵列处理则涵盖波束成形、空间滤波和方位估计等经典问题。

## 核心问题

- **水声 MIMO 信道建模**：空间相关性、阵列几何影响
- **MIMO 检测**：在高维信号空间中的最优和次优检测
- **波束成形**：利用阵列增益抑制干扰、提升信噪比
- **空时编码**：联合空间和时间域的编码设计
- **大规模 MIMO 在水声中的可行性**：受限于换能器尺寸和间距
- **阵列校准**：水下阵列的位置不确定性和相位误差

## 关键技术

- 空间复用（V-BLAST、D-BLAST）
- 空时编码（STBC、STFC）
- 自适应波束成形（MVDR、LCMV）
- MIMO 检测（ML、MMSE、球形译码）
- MIMO-OFDM 和 MIMO-OTFS
- 消息传递 MIMO 检测（AMP、EP）
- DOA 估计（MUSIC、ESPRIT）

## 实测经验（紧凑阵列 DOA 估计）

> 来源：某紧凑基阵（水平有效孔径约 1λ）受控水域实测 DOA 调试提炼。USBL 应用角度的结论见 [[usbl-positioning]]。

- **群时延选峰 vs 波束 / 载波相位是两种 DOA 模态，精度上限不同**：相关选峰取群时延会丢弃相位信息，精度被带宽经 CRLB 锁死（紧凑孔径下群时延地板约 1° 量级）；波束成形 / 载波相位模态用全孔径×全带相位，原理上可低于群时延地板——文献报道同孔径量级波束 + 反卷积校准，未校准约度级经相位校准降到**亚 0.1° 级、约一个数量级提升**（消声水域、已知目标角的理想标定条件，非现场可达指标）。关键：校准（通道相位 / 角度相关残余）只有接到波束相位模态才有数量级收益，套到群时延流形上仅能消平滑偏置。
- **小孔径的物理「模态天花板」换估计器无法突破**：当压缩脉冲主瓣宽度 >> 物理全幅 TDOA 时，各通道真时差挤在同一宽主瓣的近平顶（相邻样点差异极小），测时由噪声 / 多径而非几何决定，再经小孔径几何放大后表现为显著方位误差。这是阵列设计层（孔径 / 带宽）瓶颈：在该孔径 / 带宽下仅更换选峰估计器观察不到突破——误差由模态而非估计器决定；要进一步须换模态（波束 / 载波相位）或加孔径 / 带宽。
- **实信号匹配滤波直接取 `abs` 检峰会被载波假象锁峰**：`abs(实数 MF 相关)` 在 2·fc 处振荡，argmax 锁载波瓣而非包络主瓣，制造纯算法性的逐脉冲时延抖动与方位野值（可达几十 μs / 环形标准差几十度，是**方法假象**而非物理现象）。须改用 **hilbert 解析包络**再检峰；对包络（非带限、主瓣近平顶）做亚样点应用**抛物插值**而非 sinc（sinc 在平顶上退化到整数样点级）。这是带通信号（fc >> B）TDOA / DOA 前端的通用陷阱。
- **GCC-PHAT 本质是预白化，窄占带时抬噪**：`./(abs+eps)` 压平幅度谱、锐化时域主峰、只留延迟相位 ramp；代价是信号占带很窄（可低至个位数百分比）时大部分带外噪声被一起抬成地板。限带 PHAT 易 **cycle-skip**（结果稳但方位整体错位、物理自洽量 |u| 偏离 1），须用全带；含未解调载波的带通信号应先取包络或退到基带再做相关。注意取包络并非普遍增益——在已做多脉冲中位的 GCC 流程上反而可能退化，只针对未解调载波直接 PHAT 的 cycle-skip 场景。
- **鲁棒多脉冲共识有失效边界**：跨脉冲共识 / 中位能清除随机性整数选峰错（野值），但对低 SNR 下几何自洽的**系统性集体错峰**结构性失效——中位会固化错值且表现为低抖动（**低抖动 ≠ 高可信**）。突破须引入不靠多数表决的外部锚（载波相位独立锚 / 跨角度流形平滑先验 / 物理抑制多径）。
- **正交阵几何天然解耦病态维度**：当阵元位置矩阵三列正交（PᵀP 对角）时，LS 解退化为各轴独立差分。故即便某一维孔径远小于另一维、使该维角度估计被巨幅放大而不可靠，正交的另一维（如水平方位）仍可保持精度——正交阵可隔离病态维度的误差传播。
