# 超短基线定位 (USBL)

## 定义

超短基线 (Ultra-Short Baseline, USBL) 是一种水声定位技术，利用紧凑基阵（基线 < 0.5m）上多阵元间的相位差/时延差测量目标方位角 (DOA)，结合传播时延测距，确定水下目标三维位置。

## 技术链路

```
信号发射 → 信道传播 → 阵列接收 → 匹配滤波(测距) → DOA估计(测向) →
声线跟踪(声速修正) → 坐标变换(姿态+GPS) → 地理坐标输出
```

## 六层研究体系

1. **声学基阵与信号处理**：阵型设计、信号体制（窄带/宽带）、DOA 算法、抗相位模糊
2. **声速修正与声线跟踪**：有效声速估计、自适应分层声线追踪、声速剖面处理
3. **系统误差校准**：阵型误差修正、安装偏差校准（两步法）、校准航迹设计（需对称）
4. **定位解算与滤波**：单次解算、EKF/UKF 跟踪、异常值剔除
5. **多系统组合导航**：SINS/DVL/USBL 融合、LBL/USBL 异构组合
6. **硬件实现与集成**：换能器、应答器、GAPS 一体化

（来自 USBL 项目文献综述，9 篇论文提炼）

## 关键工程经验

- **基线分解算法**适用于任意几何构型的多元阵，系统误差可忽略（丁杰 2020）
- **安装校准航迹**需要足够尺度且对称，分两步：先估平均声速，再用去偏形式（杨保国 2013）
- **声线跟踪**对远距（>5km）定位至关重要，自适应分层可控制精度-计算量平衡（杨保国 2013）
- **ML 全局搜索**天然解决 d>λ/2 的相位模糊问题（USBL 项目实测）

## 实测经验：紧凑阵 DOA 估计模态选择（受控水域）

> 来源：某紧凑基阵（中心参考元 + 数个臂元，水平有效孔径约 1λ 量级）受控水域实测 DOA 调试，提炼为可复用结论。数值均为量级 / 带条件，非承诺指标。

- **校准收益取决于接入的 DOA 模态**：群时延 TDOA-LS（相关选峰取群时延）即便配合流形 / 相位校准（可显著降低全角方位 RMS），仍被带宽经 CRLB 决定的群时延地板之上的选峰 / 多径夺峰与小孔径反演放大锁死在**度级**、进不了亚度级；亚度级须改用**波束成形 / 载波相位模态**——同样的校准只有接到波束指向矩阵的相位修正才有数量级收益。机理与模态差异见 [[mimo-and-array-processing]] 实测经验段。
- **CBF 现场可实施精度优于群时延 TDOA（带条件）**：同批数据下，仅做通道静态相位校准的 CBF 跨方位泛化精度约 **1–2° 量级**，群时延 TDOA-LS 在重多径 / 远距下退化到**数十度**量级。条件：**单目标、低 / 中多径、整带相干**；近角（小于约一个波束宽）多径仍需 MVDR 类高分辨方法。（注：同一目标角重复测量下 CBF 逐 ping 抖动可忽略、不同斜距档间基本稳定——属同角稳定性，非跨工况泛化精度，两者不可混。）
- **分辨率 ≠ 精度**：紧凑孔径（D/λ≈1 量级）CBF 主瓣很宽（**数十度量级**，两等强源需间隔约半个到一个主瓣宽才能分开），但**单源测角精度仍可 <1°**。USBL 单目标定位只需精度，宽波束 CBF 即够用。
- **校准要区分「可迁移」与「条件内拟合」**：通道静态相位校准可迁移、可现场实施；按目标角索引的逐角残余校准在现场不可实施（现场不预知目标角），其乐观残差（亚度级）不作可达指标。全 N 通道流形模板匹配在单一工况内可拟合到约 1°（条件内过拟合值），换工装 / 距离即失效、不作可迁移指标。
- **可迁移校准需较密角栅格 + 多工况覆盖（设计建议）**：单工况方位精度主要受角栅格密度限制，跨工况泛化靠工况专属校准。建议方位用较密角栅格采样（远密于粗栅格）+ 多深度做俯仰宽跨覆盖 + 控多径（消声 / 深水 / in-situ）。此为有限工况实测数据的外推，正式验证前视作假设；与既有「安装校准航迹需对称、两步法」（杨保国 2013）属正交的两类校准采样设计。

> 上述实测与 [[quzhenzhao-2024-usbl-precision]]、[[yangbaoguo-2013-usbl-calibration]] 同类紧凑阵校准范式相互印证：同类阵的校准收益取决于接入的 DOA 模态。
