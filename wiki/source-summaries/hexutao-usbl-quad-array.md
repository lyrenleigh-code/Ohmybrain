## 核心观点

1. **相位模糊是传统 USBL 长程测距的首要精度瓶颈** —— 传统四元阵基元间距 d 受限于 d ≤ λ/2，单基元相位差测量精度直接决定定位精度上限 Δx/ρ = Δφ/π。当相位测量误差为度级时，传统阵定位精度最高仅能达到斜距的 0.556%，无法满足远距高精度需求。
2. **"非等距四元基阵"是八元阵的减元优化** —— 已有文献提出八元阵用长基线（L=8d）消除相位模糊、用短基线（d）提供无模糊参考，本文将其压缩到四元：保留一个短基线基元 (基元 3) 做相位解模糊，另外两个远端基元 (1, 4) 做 x/y 轴长基线测量，总基元数减半，精度保持一致。
3. **"间接计算 + 四舍五入取整"是相位解模糊的工程化做法** —— 利用短基线 ϕ₁,₃ 与长基线 ϕ₁,₂ 的理论倍数关系（8√2 倍），通过 `2π·round(8√2·ϕ₁,₃ₓ/2π) + ϕ₁,₂` 的相位补偿消除 2kπ 模糊，当角度误差 Δα ≤ 1° 时不会产生额外 2π 模糊。
4. **扩展卡尔曼滤波 (EKF) 对原始声波降噪优于低通和自适应残差** —— 论文对比三种声波降噪方法，在 SNR=16-22 dB 范围内 EKF 的相位误差稳定低于另两种；SNR=16 dB 时 EKF 相位误差 1.59% vs 低通 5.39%、自适应残差 2.68%。
5. **"8 倍精度提升"不是信号处理的功劳，而是阵型设计** —— 同信噪比下改进四元阵定位误差 ~8‰，传统阵 ~58‰，差距在任何 SNR 下都保持 8 倍左右；提升信噪比只能改善单阵内部精度，不能改变阵型决定的精度数量级。

## 主要方法

### 方法 1：非等距四元基阵几何设计（§2.1）

**传统四元阵问题**：四个基元十字正交等距分布，相邻基元间距 d ≤ λ/2 限制定位精度。

**本文阵型**（见论文图 5）：
- 基元 1、2、4 构成十字大骨架，基元 1 到 2/4 距离均为 L = 8l
- 基元 3 插入在基元 1 附近（距离 l），作为短基线相位参考
- 基元 1、3 方向上在 xoy 平面对角，角度为 π/4
- x 方向有效间距 d = l·cos(π/4)，y 方向 d = l·sin(π/4)

**关键几何关系**：
- L = 8l ⇒ ϕ₁,₂ ≈ 8√2·ϕ₁,₃ₓ（见式 15）
- 因为相位测量区间为 [-π, π]，当 |ϕ₁,₃ₓ| > π/(8√2) 时 ϕ₁,₂ 进入多值区间

### 方法 2：基于短基线参考的相位解模糊（§2.2）

**相位真实值表达式**：
```
ϕ₁,₂ = ϕ₁,₂₀ + 2kπ,  k ∈ {0, 1, 2, 3, ...}
```

**解模糊公式**（论文式 17）：
```
ϕ₁,₂ᶜ = 2π · round( 8√2 · ϕ₁,₃ₓ / 2π ) + ϕ₁,₂
ϕ₁,₄ᶜ = 2π · round( 8√2 · ϕ₁,₃ᵧ / 2π ) + ϕ₃,₄
```

其中 `round(·)` 为四舍五入取整。补偿后 ϕ₁,₂ᶜ、ϕ₁,₄ᶜ 不再有 2kπ 模糊，可直接用于长基线高精度测向。

**误差容忍度分析**（§2.3）：
- 角度测量 α 存在 Δα 误差时，由三角展开得：`Δϕ̂₁,₂ = -16πd·sin(α₁,₃ₓ)·Δα / λ ≤ 8π·Δα`
- 当 Δα = 1° 时，Δϕ̂₁,₂ ≤ 0.4386 < 2π，不产生额外模糊
- **结论**：算法对角度测量偏差具有良好鲁棒性

**精度上界**（式 18）：
```
Δx/ρ ≥ Δϕ₁,₂ · λ / (2π · 8d) = Δϕ₁,₂ / (8π)
```
当相位误差单位为度时，新基阵精度最高可达斜距的 0.0695%（相比传统阵 0.556%，提升 8 倍）。

### 方法 3：扩展卡尔曼滤波声波降噪（§3.1）

**对比实验**：采样率 2 kHz，SNR 从 16 dB 调至 22 dB，对比三种降噪方法的相位误差：

| SNR (dB) | 低通滤波 (%) | 自适应残差 (%) | 扩展卡尔曼 (%) |
|---------|-------------|---------------|---------------|
| 16 | 5.3937 | 2.6845 | **1.5905** |
| 18 | 2.1543 | 0.4423 | **0.3197** |
| 20 | 0.8875 | 0.2008 | **0.1915** |
| 22 | 0.3025 | 0.0922 | **0.0845** |

EKF 在各 SNR 下相位误差均最低，尤其在 SNR < 20 dB 的低信噪比区间改善最显著。仿真中后续采用 EKF 作为前置滤波器。

### 三种阵型定位误差对比（§3.2 表 3）

单位：‰（相对斜距的千分比）

| SNR (dB) | 传统阵 σ₁ | 改进四元阵 σ₂ | 改进八元阵 σ₃ |
|---------|----------|--------------|---------------|
| 15 | 59.35 | 8.397 | 8.396 |
| 20 | 58.98 | 8.202 | 8.200 |
| 25 | 57.25 | 8.187 | 8.186 |
| 30 | 55.23 | 7.235 | 7.321 |

**关键观察**：
- 改进四元阵精度与改进八元阵完全一致（差异 < 0.02‰）
- 改进阵相对传统阵精度提升约 **7 倍**（15 dB 时 59.35/8.397 ≈ 7.07）
- 信噪比从 15 dB 提升到 30 dB，改进阵精度仅改善 14%，传统阵改善 7%，**阵型优化的收益远高于 SNR 优化**
- 斜距 8000 m 条件下，改进阵定位误差 ~66 m（8.2‰ × 8000 m），仍较粗糙但已接近传统四元阵 470 m 量级的 1/7

## 对 Ohmybrain / USBL 项目的启发

### 对 USBL 项目的直接借鉴（项目视角）

1. **阵型优化优先级高于信号处理优化**：论文表 3 清晰展示阵型设计带来 ~8 倍精度提升，而 SNR 从 15 dB 提升到 30 dB 只有 14% 改善。USBL 项目如果卡在精度瓶颈，先检查阵型几何而非调参 EKF 参数。
2. **"短基线参考 + 长基线测量"的解模糊思路可复用**：USBL 项目的五元圆阵中，若选取 1 条短基线（相邻阵元）和其他长基线组合，同样可用 `round(k·ϕ_short/2π) + ϕ_long` 套路消除相位模糊，避开全局 ML 搜索的高计算代价。
3. **角度测量容忍度是工程落地的关键指标**：论文 §2.3 推导的 Δα ≤ 1° 不产生 2π 额外模糊是一个非常实用的工程判据，USBL 项目的解模糊算法也应建立类似的"角度鲁棒窗口"。
4. **EKF 对 SNR 低于 20 dB 场景改善显著**：USBL 项目若遇到浅水多径、远距低 SNR 场景，EKF 前置降噪是成本低廉的精度提升手段。
5. **非等距阵型 vs 等距阵型的取舍**：四元非等距阵相比八元等距阵省一半基元，但代价是基元 3 的位置对声学辐射中心一致性要求更高（参考丁杰 2020 的阵型误差修正）。USBL 项目若要走"少元高精度"路线，必须同步做阵型误差标定。

### 跨项目（Hub 层）价值

6. **"长短基线混合"是宽频解模糊的通用拓扑**：这一思路不限于 USBL——雷达干涉仪、GNSS 载波相位测量、MIMO 阵列 DOA 估计都可用"短基线提供无模糊参考 + 长基线提供高精度"的混合阵型。Hub 应将其抽象为"宽频解模糊阵型设计"模式。
7. **阵型设计作为"硬件层面的算法"**：论文示范了一种思维方式——把相位模糊问题从算法层（ML/MUSIC/分类器解模糊）下推到阵型层（几何约束自然消除模糊）。这是"把问题移到更简单的层次解决"的工程哲学。
8. **"改进八元阵 → 改进四元阵"的减元路径**：当已有复杂阵型 baseline 时，减元优化往往比增元优化更有工程价值（成本、体积、标定难度均降低）。这条经验可迁移到 UWAcomm 的接收机阵元数设计、UWAnet 的节点部署密度设计。
9. **仿真中控制变量法的规范示范**：论文刻意"只考虑相位测量误差"排除斜距测量等其他误差源，才能纯净地验证阵型优化的贡献。这是跨项目仿真方法论的良好案例。
10. **电网水下电缆定位是 USBL 的一个新兴行业应用**：论文背景是国家电网舟山项目（海底输电电缆定位），这一应用场景在学术文献中较少见，但工程需求明确（远距、高可靠、低成本），可作为 USBL 项目的潜在落地方向。
