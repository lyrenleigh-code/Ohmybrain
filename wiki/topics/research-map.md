# 研究全景地图

## 概述

本研究以**水声通信**为核心，围绕声波在水下环境中的可靠高速信息传输展开。论文库共计约 3179 篇论文，覆盖从数学工具到系统实现的完整技术栈。研究体系以水声通信系统为主干，以信道估计与均衡、消息传递算法为方法论核心，以信号处理和数学优化为理论基础，并持续借鉴移动通信领域的成熟技术，形成了一个层次清晰、交叉紧密的研究网络。

## 方向间关系

### 核心依赖链

```
数学与优化 → 信号处理基础 → 信道估计与均衡 → 水声通信系统
```

这是研究体系中最基本的依赖链：数学工具支撑信号处理理论，信号处理理论支撑信道估计与均衡算法，信道估计与均衡是水声通信接收机的核心。

### 方法论桥梁

**消息传递与因子图**扮演着关键的方法论桥梁角色：
- 将信道估计、均衡、检测、译码统一在概率图模型框架下
- AMP 系列算法跨越 MIMO 检测和信道估计
- Turbo 原理连接均衡与信道编译码
- 文件夹交叉证据：MessagePassing ↔ AMP (13篇)，Equalization ↔ Turbo (13篇)

### 技术迁移路径

```
移动通信 ──→ 水声通信
  (OFDM, MIMO, Turbo编码等成熟技术)
      │
      └──→ 需要针对水声信道特点进行大幅改造
           （长时延扩展、大多普勒、极低带宽）
```

### 新兴交叉

- **OTFS + 水声通信**：OTFS 调制天然匹配水声信道的双色散特性，是近年热点
- **消息传递 + MIMO**：IRC-MIMO-AMP 将近似消息传递应用于 MIMO 检测
- **机器学习 + 水声通信**：目前积累最少（2篇），但深度展开（Deep Unfolding）将消息传递算法与深度学习结合，是重要发展方向
